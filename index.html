Game Sederhana Menembakkan Peluru Ke Musuh
Informasi Game
  • Gunakan Tombol W Untuk Maju Dan Tombol S Untuk Mundur.
  • Gunakan Tombol A Untuk Bergerak Ke Kiri Dan Tombol D Untuk Bergerak Ke Kanan.
  • Tekan Spasi Untuk Menembakkan Peluru Ke Musuh.
  • Jika Musuh Terkena Peluru Maka Akan Mendapat Skor 1.
